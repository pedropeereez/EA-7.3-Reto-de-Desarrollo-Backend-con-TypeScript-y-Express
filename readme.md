# API de Gestión de Jugadores de Fútbol

Este proyecto es una API REST desarrollada con Node.js, TypeScript y Express que conecta con una base de datos MySQL para gestionar un catálogo de futbolistas.

## Requisitos Previos

* Node.js instalado.
* Servidor MySQL (XAMPP, WAMP o instalación local).
* Gestor de paquetes npm.
